
# Personal Expense Analysis Dashboard (Excel Project)

## 📌 Objective
To analyze personal spending patterns using Microsoft Excel and derive meaningful insights from transaction data.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line)
- Slicers

## 📊 Analysis Performed
- Category-wise spending analysis
- Payment method analysis
- Daily spending trend analysis
- High vs Low spending classification

## 🧠 Key Insights
- Shopping and Bills dominate total expenses  
- Card is the most used payment method  
- High-value transactions contribute over 50% of total spend  
- Spending pattern is irregular with noticeable spikes
